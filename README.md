# 鸿蒙简便工具包

为鸿蒙应用开发提供简便工具包，简化开发流程，提高开发效率

## 安装
`ohpm install  @yeluzi666/toolkit_core`

## 工具类说明

- DateUtils 日期工具
  - 日期格式化
  - 相对时间计算
  - 时间差计算
  - 时长格式化
  - 日期判断与获取

- DeviceInfo 设备信息工具
  - 获取设备型号、系统版本
  - 获取品牌、制造商、设备类型
  - 获取SDK版本与设备标识

- JsonConverter Json工具
  - JSON与对象互转
  - 安全解析与深拷贝
  - JSON合法性校验

- MathKit Math工具
  - 数字格式化与百分比计算
  - 随机数生成
  - 数值范围限制
  - 四舍五入与取整
  - 数组统计计算

- StringUtils 字符串工具
  - UUID生成
  - 格式校验（邮箱、手机号）
  - 数据脱敏处理
  - Base64编解码
  - 字符串处理（判空、截取、去HTML标签等）

- Logger 日志工具
  - 提供debug、info、warn、error等级别的日志输出

- PreferenceUtil 本地存储工具
  - 提供键值对存储功能
  - 支持数组类型的存储与操作
